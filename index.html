Word building rules used
Stage 3 reference
  • Prefix sound changes (assimilation): some prefixes change to make pronunciation easier (in + possible → impossible, con + pose → compose, ad + tract → attract).
  • Drop the final e before a vowel suffix (create + ing → creating).
  • Keep the final e before a consonant suffix (care + ful → careful).
  • Change y to i before adding most suffixes (apply + ed → applied). Do not do this for ing, and do not do this if there is a vowel before the y (pay + ment → payment).
  • Double the final consonant when a short vowel is followed by one consonant and the suffix begins with a vowel (run + ing → running).
  • Change ie to y before ing (die + ing → dying).
  • If none apply, just join the parts (re + do → redo; play + ing → playing).
